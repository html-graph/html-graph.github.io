## Remove Edge

Removes a specified edge from the canvas.

{{< code lang="javascript">}}
canvas.removeEdge("edge-1")
{{< /code >}}

### Parameters for `removeEdge`:

| Name | Type | Description                      | Required |
|------|------|----------------------------------|----------|
| id   | any  | Identifier of the edge to remove | yes      |
